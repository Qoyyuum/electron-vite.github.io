Plugins ​

  • vite-plugin-electron makes the integration of Electron With Vite very easy! Its main purpose is to build the Main process and Preload scripts, and to start and hot restart the Electron App when the Main process and Preload scripts are built.

    It also provides a full JavaScript API to make it easy to use anywhere.

  • vite-plugin-electron-renderer makes it possible to support building Electron Renderer process by modifying some of the necessary Vite default configs.

    In addition it allows some npm packages developed specifically for Node.js, especially C/C++ native modules, to be built properly by Vite to be available to the Renderer process.

中文

  • vite-plugin-electron 使得 Electron 与 Vite 的集成变得十分简单！它的主要作用是构建主进程与预加载脚本，并且当主进程与预加载脚本构建完成时启动、热重启 Electron App。

    除此之外它还提供全量的 JavaScript API 可以很方便的在任何地方使用它。

  • vite-plugin-electron-renderer 通过修改一些必要的 Vite 的默认配置使其能够支持构建 Electron 渲染进程。

    此外它使得一些专门为 Node.js 开发的 npm 包尤其是 C/C++ 本地模块，可以被 Vite 正确构建以提供给渲染进程使用。

Basic usage ​

ts
// vite.config.ts
import electron from 'vite-plugin-electron'

export default {
  plugins: [
    electron({
      entry: 'electron/main.ts',
    }),
  ],
}

Restart And Reload ​

ts
// Restart and Launch App
electron({
  entry: 'electron/main.ts',
  onstart(args) {
    args.startup()
  },
})

// Reload the Renderer process, for Preload scripts
electron({
  entry: 'electron/preload.ts',
  onstart(args) {
    args.reload()
  },
})

Code split ​

vite-plugin-electron allows for very flexible code splitting, it can be built using passing arrays, or using Vite's built-in multi-entry build.

中文

vite-plugin-electron 可以进行十分灵活的代码拆分，它可以使用传递数组的形式构建，或者使用 Vite 内置的多入口 构建。

ts
// Use array
electron([
  {
    // Main-Process entry file of the Electron App.
    entry: 'electron/main.ts',
  },
  {
    entry: 'electron/main-chunk.ts',
  },
])

// Use Vite multi-entry
electron({
  entry: {
    // Main-Process entry file of the Electron App.
    main: 'electron/main.ts'
    'main-chunk': 'electron/main-chunk.ts',
  },
})

C/C++ Native ​

  • Use in Main process
ts
// First way
electron({
  entry: 'electron/main.ts',
  vite: {
    build: {
      rollupOptions: {
        // Need to put sqlite3 into dependencies
        external: ['sqlite3'],
      },
    },
  },
})

// Second way
import native from 'vite-plugin-native'
electron({
  entry: 'electron/main.ts',
  vite: {
    plugins: [
      // Use vite-plugin-native
      native(/* options */),
    ],
  },
})
